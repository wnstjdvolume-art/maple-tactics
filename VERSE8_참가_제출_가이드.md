# 🚀 Verse8 / MSU 참가 & 제출 가이드 (정식 루트)

> ✅ **API 신청·KYC 필요 없음.** 아래 7단계만 하면 무료 토큰으로 제작·제출 가능.
> (제작은 `create.verse8.io`에서 바이브 코딩, 내부 AI=딥시크 무료 사용)

## 참가 7단계
1. **MSU Builder 등록** → https://msu.io/builder/join
   - 그냥 회원가입만. **API 신청은 ❌ (안 함)**
2. **Verse8 가입** → https://verse8.io/explore?referral=AS0FUC
3. **크리에이터 페이지에서 참가 신청** → https://verse8.io/msu/creator
4. **MSU Builder 계정 연결** → https://verse8.io/settings
   - ⚠️ 이걸 해야 **무료 토큰** 사용 가능
5. **제작 페이지에서 만들기** → https://create.verse8.io/
   - 유료 AI 안 쓰면 내부 **딥시크(DeepSeek)** 로 제작
6. **게임 만들고 SUBMIT (필수)**
7. **구글폼 제출** → https://docs.google.com/forms/d/1SFGnMPy1suW3CGjY5f8exKnxRkVAEWmj6ZNv9atVoQE/viewform

## 우리 준비물은 이미 다 됨 → 5번에서 바로 사용
- `VERSE8_BLUEPRINT.md` 의 **프롬프트 ①~⑧** 을 `create.verse8.io` AI에 **순서대로 붙여넣기**
- `index.html` 로컬 프로토타입 = "이렇게 만들어줘"의 살아있는 레퍼런스(재미/밸런스 검증 완료)
- 공식 캐릭터/몬스터 스프라이트는 **Verse8 안에서** 공식 에셋으로 지정(플랫폼이 제공)
- 시너지 표 / 유닛 수치 / 아이템 / 궁극기 스펙 모두 블루프린트에 정리돼 있음
