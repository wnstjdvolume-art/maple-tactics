# 🍁 MAPLE TACTICS — 인수인계서 (Agent8 / Verse8용)

> **이 레포를 git clone 하면 끝.** `MapleTactics.html` 이 **완성·검증된 정답 게임**이다.
> 이걸 그대로 재현(또는 서빙)하고, 그래픽만 MSU 공식 에셋으로 교체하면 된다.
> 로직·수치·밸런스는 절대 임의로 바꾸지 말 것.

## 0. 가장 먼저 (Agent8에게)
```
git clone 한 maple-tactics/MapleTactics.html 이 내가 원하는 완성 게임이다.
이 파일을 끝까지 읽고 동일하게 재현해라. 실시간 자동전투(requestAnimationFrame 매 프레임 simStep+draw),
즉시 승패 계산 금지. 유닛/시너지/별합성/궁극기/아이템/코스트별 외형 전부 이 파일과 동일하게.
캐릭터/몬스터 스프라이트는 "Use the Maple MCP" 로 MSU 공식 에셋을 적용해라(아래 7번 매핑 참고).
```
- `MapleTactics.html` 은 단일 파일로 **그대로 브라우저에서 실행**된다(인터넷 필요 — 스프라이트가 maplestory.io 라이브 URL). 먼저 열어서 "정답 동작"을 눈으로 확인하라.
- `gallery.html` = 캐릭터/몬스터/무기/스킬이펙트 한눈에 보기(검수용).

## 1. 게임 개요
메이플 직업(코스트=전직차수 1~5)으로 팀을 짜는 TFT식 오토배틀러. 준비단계(상점 영입→보드 배치) → 전투 시작 → **실시간 자동전투** → 정산 → 다음 라운드. 20라운드 생존 목표.
- 보드 7열×6행 (아래 3행=아군, 위 3행=적). 화폐="골드"만(공식 게임 화폐 명칭 사용 금지).

## 2. 유닛 21종 (직업/코스트/계열/역할/체력/공격력/공속/사거리)
`MapleTactics.html`의 `const UNITS` 참고. 요약:
- 1코: 검사(전사/수호) 매지션(마법/법사) 아처(궁수/원거리) 로그(도적/근접) 해적(해적/근접)
- 2코: 페이지 썬콜위자드 헌터 어쌔신 인파이터
- 3코: 나이트 프리스트 레인저 허밋
- 4코: 히어로 비숍 보우마스터 나이트로드
- 5코: 다크나이트 아크메이지 바이퍼
- 별 합성: 같은 직업 3개=별2(능력치×1.8), 별2 3개=별3(×3.2). **벤치 꽉 차도 "사면 합성되는" 구매는 허용**(중요).

## 3. 시너지 (계열5 + 역할5, 서로 다른 직업 수로 카운트)
- 전사 2/4/6 체력+200/500/1000 · 마법사 2/4 마공+25/60 · 궁수 2/4 공속+25/55% · 도적 2/4 공격+18/40% · 해적 2/3/4 체공+12/22/35%
- 수호자 2/4 체력+300/700 · 근접 2/4 공격+20/45% · 원거리 2/4 공격+20/50 · 법사 2/3 마공+35/80 · 보조 2 시작체력+20%

## 4. 전투 (★실시간, 즉시계산 금지)
`combatLoop`가 requestAnimationFrame으로 매 프레임 `simStep()`(가장 가까운 적 탐색→사거리면 공격(1/공속), 아니면 한 칸 이동) + `drawCombat()`(렌더). 한쪽 전멸/30초면 종료. 치명타 15%(도적 30%)×1.6. 패배 시 체력 -= min(18, 살아남은 적 수+floor(라운드/3)+1).

## 5. 궁극기 (마나 + 직업별 시그니처 이펙트)
공격 시 마나 +10, 가득(=22+코스트×5) 차면 자동 시전. 화면 중앙 스킬명 + 불릿타임(전투 느려지고 이펙트 재생). 직업별 vfx(`ULT_VFX`):
- 아크메이지 **메테오** · 썬콜 **체인라이트닝** · 매지션/어쌔신 **고드름(氷)** · 히어로 **검기파동** · 다크나이트 **흑염임팩트** · 비숍 **홀리노바** · 보우마스터 **화살폭풍** · 나이트로드 **표창난사** · 궁수 화살비 · 검사/로그/허밋 세비지블로우(斬) · 프리스트 제네시스 · 페이지/나이트 보호막 · 해적/바이퍼 빔.

## 6. 아이템 (조각 조합)
드롭=조각5종(검/갑옷/장갑/보주/망토). 조각 2개 합쳐 완성템 15종(`RECIPES`). 유닛에 최대 3장착, 미완성 조각 위에 끼우면 자동 합성. 조합표 버튼 제공.
**★별 승급 시 아이템 계승(필수)**: 같은 직업 3개 합쳐 승급할 때, 3유닛이 들고 있던 아이템을 모두 모아 **승급 유닛에 최대 3개 계승**하고, 초과분은 가방(`G.comps`/`G.items`)으로 반환한다. (승급 유닛을 빈손으로 만들면 안 됨 — `combineCheck` 참고)

## 7. ★스프라이트 / 코스트별 외형 (MSU 공식 에셋 매핑)
`MapleTactics.html` 하단의 설정 표(`HAIR / OVERALL2 / HAT / CAPE / WEAPON / FIVE_HAT / MOB_ID`)가 외형을 결정한다. 현재는 maplestory.io 라이브 합성 URL을 쓰며, **Verse8에서는 같은 itemId를 MSU 공식 에셋으로 매핑**하면 된다.

**코스트별 외형 사다리 (확정):**
- 1코 = 맨몸 + 기본 무기
- 2코 = +직업 모자 (전사=철투구, 마법사=뾰족모자, 궁수=와일드헌터햇, 도적=후드, 해적=반다나)
- 3코 = +직업 갑옷(계열색) + 모자, 무기 중급
- 4코 = +상급 갑옷 + 망토 + 모자
- 5코 = **보스 투구(블루 드래곤 헬멧) + 에픽 갑옷 + 망토 + 에픽 무기** (가장 화려, 크기 1.5배)
- 크기: 4코 1.25배 / 5코 1.5배. 몸통 높이 고정(무기는 옆으로 삐져나옴).

**무기 itemId (1코 기본→5코 에픽)** 및 **모자/갑옷 itemId**: `MapleTactics.html`의 `WEAPON / HAT / OVERALL2 / FIVE_HAT` 객체 그대로 사용.

**스프라이트 URL 규칙 (참고):**
- 캐릭터 합성: `https://maplestory.io/api/character/{ {"itemId":X,"region":"GMS","version":"255"} 콤마조인 }/stand1/0`
- 몬스터: `https://maplestory.io/api/GMS/255/mob/{mobId}/render/stand`
- Verse8 공식 전환 시: 위 itemId/mobId 를 **Maple MCP로 조회**해 공식 텍스처로 교체.

## 7-B. ★검수 반영 사항 (재제출 필수)
검수자 피드백 3가지가 `MapleTactics.html`에 이미 반영됨 — Agent8은 그대로 따라야 한다.
1. **영어 기본 + 한국어 선택**: 기본 언어 English, 우상단 `langBtn`(🌐)으로 한국어 토글. 구현 = `EN`/`KO` 문자열 테이블 + `t(key)` + `setLang(l)`(데이터 `.name`/`.desc`를 언어별로 교체 + 정적 UI를 id로 세팅). 시작 시 `setLang('en')`. **표시 코드는 안 건드리고 .name만 스왑**하는 방식. 유닛/몬스터/시너지 이름·도움말·버튼·오버레이 전부 번역됨.
2. **HUD 정렬**: `#topbar`에 `flex-wrap:wrap; row-gap` + `.stat{white-space:nowrap}` + `#phase{white-space:nowrap}` 로 깨짐 방지. (Agent8 버전에서 어긋나던 요소들 정리)
3. **캐릭터는 항상 옷 착용**: 1·2코도 `OVERALL2`에 기본 의상 itemId 지정(맨몸 금지). 1코=기본의상, 2코=조금 더 갖춤, 3~5코=계열 갑옷.
