아래는 우리가 완성·검증한 게임의 "정답 스펙"이다. 네가 임의로 유닛/시너지/수치를 지어내지 말고, 이 스펙 그대로 구현해라. 로직과 밸런스 숫자는 절대 바꾸지 말고, 그래픽/사운드만 메이플스토리 공식 에셋으로 사용해라. 화폐 이름은 "골드"(공식 게임 화폐 명칭 사용 금지).

# 게임: MAPLE TACTICS (메이플 직업 오토배틀러 / TFT식)
- 내 유닛 = 메이플 직업 캐릭터(코스트 = 전직 차수 1~5). 적 = 메이플 몬스터 웨이브.
- 전장: 7열 x 6행. 아래 3행 = 내 진영, 위 3행 = 적 진영.
- 흐름: 준비 단계(상점에서 영입→보드 배치) → "전투 시작" → 자동 전투 → 정산 → 다음 라운드.
- 목표: 내 체력(100)이 0이 되기 전 20라운드 생존.

## 2. 시너지 (계열 5종 + 역할 5종). 보드 위 "서로 다른 직업" 수로 카운트(같은 직업 중복=1). 해당 트레잇을 가진 유닛에게만 효과 적용.
계열:
- 전사: 2/4/6 → 추가 체력 +200 / +500 / +1000
- 마법사: 2/4 → 공격력(마법) +25 / +60
- 궁수: 2/4 → 공격속도 +25% / +55%
- 도적: 2/4 → 공격력 +18% / +40%
- 해적: 2/3/4 → 체력·공격력 +12% / +22% / +35%
역할:
- 수호자: 2/4 → 추가 체력 +300 / +700
- 근접: 2/4 → 공격력 +20% / +45%
- 원거리: 2/4 → 공격력 +20 / +50
- 법사: 2/3 → 공격력(마법) +35 / +80

(각 유닛은 [계열 1개 + 역할 1개] 두 시너지에 동시에 기여)

## 3. 별 합성(전직)
- 같은 직업 3개 = 별2 (능력치 1.8배). 별2 3개 = 별3 (3.2배). 보드 위치 유지하며 자동 합성.

## 4. 경제 / 레벨 / 상점
- 시작: 골드 6, 레벨 2.
- 라운드 정산 골드 = 기본 5 + 이자(보유 10골드당 +1, 최대 +5) + 연승/연패 보너스(2연속 +1, 3연속 +2, 5연속+ +3) + 승리 시 +1.
- 경험치: 구매(4골드 = +4xp), 라운드마다 자동 +2.
- 레벨업 필요 경험치: 레벨1→2:2, 2→3:4, 3→4:8, 4→5:14, 5→6:24, 6→7:40, 7→8:60, 8→9:84.
- 레벨 = 보드에 올릴 수 있는 유닛 수: 레벨 1~9 = 3,3,4,5,6,7,8,9,10.
- 상점 5칸. "새로고침"(2골드). 카드 클릭 = 골드 소모해 영입(벤치로).
- 상점 등장 확률(레벨별, 코스트 1/2/3/4/5 %):
  Lv1 100/0/0/0/0, Lv2 75/25/0/0/0, Lv3 55/30/15/0/0, Lv4 45/30/20/5/0,
  Lv5 35/30/25/8/2, Lv6 25/30/30/13/2, Lv7 20/25/32/18/5, Lv8 16/22/30/24/8, Lv9 12/18/28/28/14.
- 유닛 풀(공유 재고): 코스트별 보유 수량 1코18 2코14 3코12 4코10 5코9.

## 5. 전투 규칙(자동)
- 각 유닛: 매 틱 가장 가까운 적 탐색 → 사거리 안이면 공격(공격간격 = 1/공격속도), 아니면 한 칸 이동.
- 근접 사거리=1칸(인접), 원거리=3~4칸.
- 치명타: 기본 15% 확률(도적 계열은 +15% = 30%), 치명 시 1.6배 데미지.
- 전투 종료: 한쪽 전멸 또는 30초 시간초과(생존자 많은 쪽 우세).
- 패배 시 내 체력 감소 = 살아남은 적 수 + floor(라운드/3) + 1 (최대 18).
- 배치: 클릭=보드↔벤치 올리기/내리기, 드래그=정밀 배치, 우클릭=판매. 전투 속도 ×1/×2/×3 선택 버튼.

## 6. 궁극기(마나 + 직업별 시그니처)
- 각 유닛 마나: 최대치 = 22 + 코스트×5. 공격 시 +10, 피격 시 +4. 가득 차면 자동 시전 후 0으로.
- 역할별 기본 궁극기:
  - 수호자: 보호막(자신 최대체력의 30%×pow)
  - 근접: 단일 강타(공격력×pow, 가장 가까운 적)
  - 원거리: 다수 연사(가까운 N명에게 공격력×pow×0.6)
  - 법사: 광역 폭발(대상 주변 적들에게 공격력×pow)
  - 보조: 가장 체력 낮은 아군 회복(최대체력×25%×pow)
- 직업별 시그니처(이름 / 타입 / pow [/옵션]):
  히어로=히어로즈 윌/근접nuke/2.6, 다크나이트=다크 임팩트/nuke/2.4, 바이퍼=에너지 버스터/nuke/2.3,
  나이트=아이언 윌/보호막/1.4, 아크메이지=제네시스/광역aoe/1.9(반경2), 비숍=엔젤레이/회복/1.8,
  보우마스터=하이퍼 스트라이크/연사/1.3(4명), 나이트로드=쇼다운/연사/1.1(5명), 레인저=애로우 레인/연사/1.2(3명),
  프리스트=홀리 라이트/회복/1.3, 허밋=어벤져/nuke/1.9.
  (시그니처 없는 유닛은 역할 기본 궁극기 사용)
- 시전 시: 화면 중앙에 큰 스킬명 표시 + 불릿타임(전투 잠깐 느려지고 이펙트 재생) + 4·5코는 캐릭터 컷인.
- ★스킬 이펙트·사운드는 메이플 공식 에셋 사용(번개/빔/참격/빛기둥/회복 등 해당 직업 스킬에 맞게).

## 7. 아이템 (조각 조합)
- 기본 조각 5종(작은 스탯): 검 조각(공격력+12), 갑옷 조각(체력+120), 장갑 조각(공속+8%), 보주 조각(공격력+14), 망토 조각(피해감소+5%).
- 조각 2개를 합치면 완성 아이템(15종). 조합표(행+열):
  검+검=적룡도(공격력+35) / 갑옷+갑옷=이지스실드(체력+350) / 장갑+장갑=트랜스포저(공속+25%) / 보주+보주=마력의정수(공격력+50) / 망토+망토=우트가르드(피해감소+15%)
  검+갑옷=도미네이터(공격력+15,체력+180) / 검+장갑=흡혈장갑(공격력+10,흡혈20%) / 검+보주=제로의검(공격력+22%) / 검+망토=가시갑주(공격력+14,피해감소+8%)
  갑옷+장갑=질풍갑옷(체력+160,공속+12%) / 갑옷+보주=현자의로브(체력+160,공격력+18) / 갑옷+망토=강철요새(체력+220,피해감소+10%)
  장갑+보주=룬글러브(공속+15%,공격력+18) / 장갑+망토=바람막이(공속+15%,피해감소+8%) / 보주+망토=정령의부적(공격력+22,피해감소+8%)
- 드롭: 1~3라운드 보장 1개, 이후 승리 시 50%로 1개, 보스 승리 시 +1개. 드롭은 "조각" 5종 동일확률(각20%).
- 장착: 유닛당 최대 3슬롯. 조각도 캐릭터에 장착 가능하고, 캐릭터에 미완성 조각이 있는 상태에서 다른 조각을 끼우면 자동으로 완성 아이템으로 합쳐짐.

## 8. 적 / 보스 / 라이벌(PvP)
- 일반 라운드: 메이플 몬스터 웨이브(라운드 비례로 수·능력치 강해짐, 능력치 +3.5%/라운드). 몬스터 티어:
  t1 달팽이/주황버섯/리본돼지, t2 와일드보어/스텀프/파란버섯, t3 주니어네키/커즈아이/좀비버섯, t4 예티/드레이크, t5 그리피.
- 보스 라운드 5/10/15/20: 머쉬맘 → 자쿰 → 혼테일 → 발록 (거대 체력 + 호위 몬스터).
- 라이벌(PvP) 라운드 4/8/12/16/19: 다른 플레이어 보드(싱글에선 AI가 직업으로 구성한 팀)와 대전. 지면 살아남은 적 수만큼 체력 감소. (멀티 모드 확장 대비: 적 팀을 "상대 플레이어 보드"로 교체만 하면 됨)
- 몬스터 피격 시 피격 모션, 공격/스킬 시 메이플 공식 사운드.

## 9. 반려 방지(필수)
- 화폐는 "골드"만. 공식 게임 화폐 명칭 사용 금지(디스클레이머에도 언급 금지).
- 인게임/설명/태그에 "Nexon/MSU/MaplestoryU/MaplestoryN" 같은 공식인 척 표기 금지.
- 확률형 요소(상점 등장확률, 조각 드롭) 확률표를 게임 내 도움말 또는 설명에 기재.
