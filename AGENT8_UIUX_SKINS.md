아래 두 가지를 정확히 구현해라. (A) 플레이어 조작법(UI/UX) 전체, (B) 코스트별 옷차림 차등. UI 모양/배치는 내가 준 캡처 이미지를 따르고, 동작 규칙은 아래대로.

# A. 조작법 (UI/UX) — 아주 정확하게

## A-1. 상점에서 영입
- 상점 카드(5칸) **클릭** = 그 직업 영입. 골드를 코스트만큼 차감하고 **벤치 빈칸**에 추가. 영입 직후 합성 체크.
- 골드 부족하면 영입 안 되고 "골드 부족" 표시. 벤치 가득(9칸)이면 합성으로 이어지지 않는 한 영입 불가.
- **새로고침** 버튼(2골드) = 상점 5칸 다시 굴림. **경험치** 버튼(4골드 = +4xp). **전투 시작** 버튼.

## A-2. 배치 (벤치 ↔ 보드) — 클릭 토글 + 드래그
- 유닛 **클릭(짧게)** = 토글:
  - 벤치 유닛 클릭 → 보드의 빈 앞자리(아래 3행, 앞줄부터)로 **자동 배치**.
  - 보드 유닛 클릭 → 벤치 빈칸으로 **회수**.
- 유닛 **드래그** = 원하는 칸에 **정밀 배치**:
  - 드래그하는 동안 유닛 미니 아이콘(고스트)이 커서를 따라온다.
  - 놓을 수 있는 칸(내 진영=아래 3행)이 **금색 테두리로 하이라이트**.
  - 보드의 다른 유닛 위에 놓으면 **두 유닛 위치 교환**.
  - 벤치 슬롯 위에 놓으면 그 슬롯으로 이동(차 있으면 빈칸으로).
- **우클릭** = 그 유닛 판매(골드 환급, 유닛 풀에 반환, 장착 아이템/조각은 인벤토리로 회수).
- 보드 정원(=레벨) 초과 시 보드에 더 못 올림 → "보드 정원 초과" 표시.

## A-3. 아이템 / 조각
- 우측 인벤토리에 **조각**과 **완성 아이템**이 따로 보임.
- **조각 클릭 → 다른 조각 클릭** = 두 조각을 조합해 완성 아이템 생성.
- **조각/아이템 클릭 → 캐릭터(유닛) 클릭** = 그 유닛에 장착(최대 3슬롯).
  - 유닛에 미완성 조각이 이미 있는 상태에서 다른 조각을 장착하면 → 자동으로 완성 아이템으로 합쳐짐.
- **📖 조합표** 버튼 = 5×5 레시피 표 팝업.

## A-4. 기타
- **전투 속도 ×1 / ×2 / ×3** 버튼(상단). 전투 시뮬 배속.
- (선택) 키보드: D=새로고침, F=경험치, Space=전투 시작.
- **모바일 터치 대응**: 탭=클릭, 길게 누르고 끌기=드래그, 길게 누르기=판매(또는 판매 버튼).
- 상단바: ❤️체력 | 💰골드 | ⭐레벨(현재xp/필요xp, 보드정원) | ◆라운드 | 상태(준비/전투/정산) | 속도버튼.
- 하단 도움말 한 줄: "클릭=올리기/내리기 · 드래그=정밀배치 · 우클릭=판매 · 카드클릭=영입".

# B. 코스트별 옷차림 차등 (★공식 메이플 캐릭터 합성 이용)

핵심 아이디어: 메이플 캐릭터는 "신체(스킨) + 헤어 + 얼굴 + 장비(모자/상의/하의/무기/망토/탈것)"를 **레이어로 합성**해 그린다. 그러니 **계열(직업군)** 은 무기·테마로, **코스트(전직 차수)** 는 "장비를 얼마나 입었나"로 구분한다.

## 계열 = 무기/테마 (같은 계열은 같은 무기류)
- 전사 = 검/갑옷, 마법사 = 지팡이/로브, 궁수 = 활, 도적 = 단검/표창, 해적 = 너클/총.

## 코스트 = 장비 레이어 (옷을 점점 더 입음)
- **1코스트 = 맨몸**: 신체(스킨) + 헤어 + 얼굴만. 장비 거의 없음(기본 무기만). 가장 헐벗은 초보 모습.
- **2코스트 = 투구만**: 위 + **모자/투구(hat)** 1개 착용.
- **3코스트 = 상하의 포함**: 위 + **상의(top) + 하의(bottom)** (또는 전신옷 overall)까지 갖춰 입음.
- **4코스트 = 캐쉬 스킨까지**: 위 + **화려한 캐쉬/프리미엄 스킨 장비**(고급 코디 + 망토 등). 한눈에 "비싼 유닛".
- **5코스트 = 탈것 탑승**: 위 + **탈것(mount/라이딩)** 에 올라탄 모습. 가장 압도적인 비주얼.

## 구현 지시 (MSU 공식 에셋으로)
- 각 직업 유닛을 MSU/메이플 **공식 캐릭터 + 장비 에셋을 레이어 합성**해서 렌더해라(AI로 새로 생성 금지).
- 같은 계열이라도 코스트가 오를수록 위 표대로 **장비 레이어를 추가**해서, 1코는 헐벗고 5코는 탈것 탄 화려한 모습이 되게 한다.
- 예: 전사 계열 — 검사(1코)=맨몸+검, 페이지(2코)=투구+검, 나이트(3코)=갑옷 상하의+검, 히어로(4코)=캐쉬 갑옷+망토+대검, 다크나이트(5코)=탈것 탑승+창.
- 별(★)은 옷이 아니라 오라/이펙트/크기로 구분(별2=하늘빛 오라, 별3=황금 오라+입자, 살짝 커짐).
